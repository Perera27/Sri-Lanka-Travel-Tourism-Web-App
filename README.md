# Sri-Lanka-Travel-Tourism-Web-App

# Django Prototype

## Setup (Local)
1. Create & activate venv:
   python3 -m venv venv
   source venv/bin/activate

2. Install dependencies:
   pip install -r requirements.txt

3. Run migrations:
   python manage.py migrate

4. Run server:
   python manage.py runserver
